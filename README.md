# koa2-st
:sparkles: koa2 start kit tool

## Note
- Node.js 7.6+ 
- Babel-es2017 (无需对 Node.js 7.x 已支持的特性转码)

## Installation
```
sudo npm i koa2-st -g
```

## Usage
全局安装koa2-st后, 使用`kst`命令即可在当前目录生成服务端程序  
地址: ${workspaceDir}/${projectName}
```bash
kst

cd koa2-st && npm i #建议使用 nrm 管理 npm 安装源
```
